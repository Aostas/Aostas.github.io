196pin示例 ​

一、执行效果 ​


二、初始化 ​

[00]选择工具，读取项目的配方文件。
[01]选择工具，全局变量初始化。
[02]选择工具，获取点云
[03]选择工具，将点云复制一份到IM1（IM0不动后面可以再次使用）。

三、左半部分处理 ​

1.图像定位 ​

[04]选择工具，将点云进行90度的旋转

[05-08]选择工具，选择4个边界的pin针，输出其中心点的坐标

[09]选择工具，计算出4个点的x和y坐标的平均值，作为左半部分的中心点

[10]选择工具，选择上1步计算出来的中心点为原点，调整点云的X和Y位置

2.拟合平面 ​

[11]选择工具，选择左半部分底面比较平坦的区域，合并成一个区域

[12]选择工具，选择上1步计算出的区域，拟合平面

3.裁切滤波 ​

[13]选择工具，选择要进行测量的pin的区域，并输出到新的IM2中进行处理

[14]选择工具，去除pin周围的一切干扰噪点

4.确定测量起点 ​

[15]选择工具，选择pin最中间一行，根据所有pin生成一条直线

[16]选择工具，选择上1步输出的直线，调整点云X和Y的位置

[17]选择工具，选择pin的最左边一列作为起始列，找出每一行的第一个点

5.循环测量每个pin ​

lua
--判断循环次数
smallpcbPinLoop = GetIntVariable("SmallpcbPinLoop") + 1
SetVariable("SmallpcbPinLoop", smallpcbPinLoop)

if smallpcbPinLoop == 21 then
    smallPcbColumn = GetIntVariable("SmallPcbColumn") + 1
    SetVariable("SmallPcbColumn", smallPcbColumn)
    if smallPcbColumn < 5 then
        Jump("loop Column")
    else
        Jump("None")
    end
elseif smallpcbPinLoop < 21 then
    Jump("loop pin")
end
