SDN40 Mating端测量示例 ​

一、执行效果 ​




二、初始化 ​

[00] 选择工具，对全局变量进行初始化。
[01] 选择工具，获取点云。
[02] 选择工具，将点云复制一份到IM1（IM0不动后面可以再次使用）。

三、Mating端右半部分处理 ​

1.图像定位 ​

[03]选择工具，找到定位框，输出边框-Y方向的一条直线

[04]选择工具，将Y方向的位置调整到上一步输出的直线上

[05-10]选择工具，找到固定物体的边框，并将边框的各条边输出

[11]选择工具，获取前6步生成的线的坐标，并计算其中心的坐标，作为Mating端右半部分的中心

[12]选择工具，将上1步生成的中心点作为新的原点，调整X和Y的位置

2.拟合平面 ​

[13-14]选择工具，选择Mating端右半部分底面4块较平整的区域进行合并

[15]选择工具，根据上1步生成的区域，拟合成一个平面

3.裁切滤波 ​

[16]选择工具，选择要进行测量的pin的区域，并输出到新的IM2中进行处理

[17-18]选择工具，去除pin周围的一切干扰噪点

4.位置粗判断 ​

[19-21]选择工具，找到3排pin的第一个pin的中心点

[22]选择工具，判断找到的3个pin的位置是否在设定范围内，并将结果显示在IM2的窗口中

lua
x1, y1, z1 = GetPoint(0)
x2, y2, z2 = GetPoint(2)
x3, y3, z3 = GetPoint(4)

reduceresult1 = math.abs(x2 - x1)
x1mind = x1
y1mind = (y1 + y2) / 2
z1mind = z1 - 10

if reduceresult1 > 5.58 or reduceresult1 < 4.78 then
    println("nearpinresult1NG")
    DrawOverlay(2, x1mind, y1mind, z1mind, 18, "#ff0000", "nearpinresult1NG")
else
    println("nearpinresult1OK")
    DrawOverlay(2, x1mind, y1mind, z1mind, 18, "#00ff00", "nearpinresult1OK")
end

reduceresult2 = math.abs(x3 - x2)
x2mind = x2
y2mind = (y2 + y3) / 2

    println("nearpinresult2NG")

else
    println("nearpinresult2OK")

[23-26]重复19-22的方法，在另一端找到3个边缘点，并判断位置是否正确

5.确定测量起点 ​

[27]选择工具，选择底层的所有pin脚。
[28]选择工具，更加精确的选择要测量的区域，也可以手动滤除一些较大难以过滤的点，输出到IM3窗口

[29-32]选择工具，选择新区域中最左边一列，和中间一行，并输出两条直线，脚本计算两条线的交点，作为新原点进行位置调整


[33-34]选择工具，pin针最上边一排作为起点，找到每一列的第一个pin的位置，并保存在全局变量中

6.循环测量每个pin ​

[35-37]选择工具，标明循环的列和行，并在每列开始时，通过脚本获取该列的起始点位置

lua
SetVariable('SmallpcbPinLoop', 0)

smallPcbColumn = GetIntVariable('SmallPcbColumn')
x, y, z = GetPoint('PCB_Start' .. (smallPcbColumn + 1))
SetPoint('SmallpcbColumnStart', x, y, z)

println('SmallpcbColumnStart = ', x, y, z)
SetVariable('SmallpcbPinLoop', 0)

smallPcbColumn = GetIntVariable('SmallPcbColumn')
x, y, z = GetPoint('PCB_Start' .. (smallPcbColumn + 1))
SetPoint('SmallpcbColumnStart', x, y, z)

println('SmallpcbColumnStart = ', x, y, z)

[43]选择工具，循环次数判断，一列结束时，跳转到下一列，所有列都结束后，跳转到None标记，代表此半边pin检测完成

lua
smallpcbPinLoop = GetIntVariable("SmallpcbPinLoop") + 1
SetVariable("SmallpcbPinLoop", smallpcbPinLoop)

if smallpcbPinLoop == 18 then
    smallPcbColumn = GetIntVariable("SmallPcbColumn") + 1
    SetVariable("SmallPcbColumn", smallPcbColumn)
    if smallPcbColumn < 3 then
        Jump("loop Column")
    else
        Jump("None")
    end
elseif smallpcbPinLoop < 18 then
    Jump("loop pin")
end
smallpcbPinLoop = GetIntVariable("SmallpcbPinLoop") + 1
SetVariable("SmallpcbPinLoop", smallpcbPinLoop)

if smallpcbPinLoop == 18 then
    smallPcbColumn = GetIntVariable("SmallPcbColumn") + 1
    SetVariable("SmallPcbColumn", smallPcbColumn)
    if smallPcbColumn < 3 then
        Jump("loop Column")
    else
        Jump("None")
    end
elseif smallpcbPinLoop < 18 then
    Jump("loop pin")
end
