SRR5示例 ​

一、执行效果 ​


二、初始化 ​

[00-01]选择加载点云工具，获取点云。
[02]选择Lua脚本语言工具，全局变量初始化，创建待接受数据的csv文件。
[03]选择3D裁切预处理工具，将点云复制一份到IM1（IM0不动后面可以再次使用）。

三、PCB面处理 ​

1.拟合平面 ​

[04]选择3D变换预处理工具，将点云进行90度的旋转。


[05]选择3D区域工具，选择中心部分底面比较平坦的区域，合并成一个区域。


[06]选择3D平面工具，选择上1步计算出的区域，拟合平面。


2.图像定位 ​

[07]选择3D斑点工具，选择右侧凸点，输出其中心点坐标。


[08]选择3D位置调整工具，选择上1步计算出来的中心点为原点，调整点云的X和Y位置。


3.裁切滤波 ​

[09]选择3D裁切预处理工具，选择要进行测量的pin的区域，并输出到新的IM2中进行处理。


[10-12]连续3次3D离群点滤波工具，去除pin周围的一切干扰噪点。


4.确定测量起点 ​

[13]选择3D斑点工具，选择pin的最右边一列作为起始列，找出每一行的第一个点。


5.循环测量每个pin ​

[14-17]选择标记工具，记录每一行和每一列的循环标记，并通过脚本获取每一行的起始点坐标。

lua
--判断循环次数

if smallpcbPinLoop == 4 then
    smallPcbColumn = GetVariable("SmallPcbColumn") + 1
    SetVariable("SmallPcbColumn", smallPcbColumn)
    if smallPcbColumn < 2 then
        Jump("loop Column")
    else
        fprint("./pcb.csv","\n")
        Jump("None")
    end
elseif smallpcbPinLoop < 4 then
    Jump("loop pin")
end
