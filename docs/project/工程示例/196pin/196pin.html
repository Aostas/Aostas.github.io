196pin示例 ​

一、执行效果 ​


二、初始化 ​

1.预备工作 ​

[*]配方加载，点击第二行最后一个小图标后，打开配方界面。


[**]右击左侧配方列表可创建，添加或删除已有配方。


[***]打开配方后，可在右侧直接看到配方内容，也可以在右侧直接修改配方内容。


2.初始化工作 ​

[00]选择Lua脚本语言工具，全局变量初始化，创建待接受数据的csv文件。
[01]选择加载点云工具，获取点云。
[02]选择3D裁切预处理工具，将点云复制一份到IM1（IM0不动后面可以再次使用）。

三、左半部分处理 ​

1.图像定位 ​

[03]选择3D变换预处理工具，将点云进行90度的旋转。


[04-07]选择3D方形探针工具，选择4个边界的pin针，输出其中心点的坐标。


[08]选择Lua语言脚本工具，计算出4个点的x和y坐标的平均值，作为左半部分的中心点。

lua
x1, y1, z1 = GetPoint(6)   --GetPoint(i)表示读取寄存器中序号为i的点的x,y,z坐标的信息。
x2, y2, z2 = GetPoint(4)
x3, y3, z3 = GetPoint(2)
x4, y4, z4 = GetPoint(0)

x = (x1+x2+x3+x4) / 4
y = (y1+y2+y3+y4) / 4
z = (z1+z2+z3+z4) / 4

PushPoint(x, y, z)    --PushPoint(x,y,z)表示将点x,y,z输出到寄存器序号为0的位置上。
x1, y1, z1 = GetPoint(6)   --GetPoint(i)表示读取寄存器中序号为i的点的x,y,z坐标的信息。

x3, y3, z3 = GetPoint(2)

y = (y1+y2+y3+y4) / 4
z = (z1+z2+z3+z4) / 4

PushPoint(x, y, z)    --PushPoint(x,y,z)表示将点x,y,z输出到寄存器序号为0的位置上。

[09]选择3D位置调整工具，选择上1步计算出来的中心点为原点，调整点云的X和Y位置。


2.拟合平面 ​

[10]选择3D区域工具，选择左半部分底面比较平坦的区域，合并成一个区域，并将区域输出到寄存器中。


[11]选择3D平面工具，从寄存器中选择上1步计算出的区域，拟合平面作为下面计算的零平面。


3.裁切滤波 ​

[12]选择3D裁切预处理工具，选择要进行测量的pin的区域，并输出到新的IM2中进行处理。


[13]选择3D离群点滤波工具，去除pin周围的一切干扰噪点。


4.调整位置确定方向 ​

[14]选择3D方形探针工具，选择pin最中间一行，根据所有pin生成一条直线。


[15]选择3D位置调整工具，选择上1步输出的直线，调整点云X和Y的位置。

5.Pin针匹配（需配合配方） ​

[16]选择Pin针匹配工具，选择输入图像，pin针起始编号。


选择pin的最左边一列作为起始列，找出每一行的第一个点。


选择循环的行数和列数，框选pin针的box大小，需要测量的内容：高度，间隔，并根据配方进行NG,OK判断并将结果输出在图像上。选择高度测量的参数。


将各pin针结果输出到创建的变量中。



三、右半部分处理 ​

[17-34]此部分处理逻辑和左半部分的一模一样，请参考上一节的流程。

四、结果输出 ​

[35]pin针匹配工具输出的结果默认为json格式，lua脚本解析输出的变量，进行结果读写。

lua
pinStr1 = GetVariable("pin_1_105")
pinJson1 = ParseJson(pinStr1)    --解析Json

write = ""
for i=0,104,1 do         --将每一个pin针的数据写入csv文件中
    write = write .. string.format("%.3f,", pinJson1.result[i].PinPositionX)
    write = write .. string.format("%.3f,", pinJson1.result[i].PinPositionY)
    write = write .. string.format("%.3f,", pinJson1.result[i].PinHight)
    write = write ..  " ,"
end

pinStr2 = GetVariable("pin_106_111")
pinJson2 = ParseJson(pinStr2)

for i = 0,5,1 do
    write = write .. string.format("%.3f,", pinJson2.result[i].PinPositionX)
    write = write .. string.format("%.3f,", pinJson2.result[i].PinPositionY)
    write = write .. string.format("%.3f,", pinJson2.result[i].PinHight)
    write = write .. " ,"
end

    write = write .. string.format("%.3f,", pinJson3.result[i].PinHight)
    write = write .. " ,"
end

fprint("./196pin.csv", write)
