仅介绍常用组件

图表组件 ​

1.公共配置 ​

a.渲染器设置：

全局控制所有图表渲染设置，支持两种格式，分别为 svg、canvas

当前 控制当前所选组件渲染设置，具有三个选项，分别为 svg、canvas、inherit

svg 渲染在缩放场景下具有更好表现

canvas 渲染在数据量较大、较多交互时具有更好表现

inherit 为使用全局渲染设置

b. 图列设置 ​

2.柱状图&&折线图 ​

柱状图与折线图独有设置为 X 轴设置、Y 轴设置、图列设置、折线(图柱)设置

a. X 轴设置&&Y 轴设置 ​

b.图柱/折线设置 ​

此部分主要是对折线/圆柱进行一些显示上的设置，数据绑定 stduio 全局变量中的 number 类型参数，绑定参数后预览页即可展示 studio 数据

柱状图&折线图最多显示 10 条数据，多余数据会自动收缩，点击折线图拐点/柱状图柱子即可展开相应数据

柱状图&折线图最多保存 2000 条或两小时内数据，保存数据量可在系统设置中设置

2.饼图 ​

饼图独有设置为饼图配置与饼图数据

a.饼图配置

b.饼图数据

参考折线图/柱状图绑定 studio 参数部分

信息类组件 ​

样式部分就不一一缀叙

1.日志&&文字组件 ​

日志组件支持仅支持绑定 string 类型组件，其最大保存数据量支持在全局设置中修改

文字组件支持绑定 string、number 类型参数

2.标签选择器组件 ​

标签选择器组件支持绑定当前画布内除标签选择器外所有组件，用来切换绑定组件的显示与隐藏。

标签选择器组件仅支持绑定单个组件，若想绑定多个组件，请将组件进行分组，绑定分组

列表组件 ​

样式部分就不一一缀叙

1.基础分页表格 ​

基础分页表格组件支持绑定 number 类型参数，用来展示 studio 运行数据，其最大保存数据量支持在全局设置中修改

2.配方表格

配方表格用来展示 studio 配方，用户拖进画布后无需进行过多设置即可在预览页正常使用

刷新按钮：studio 修改配方表格后，需刷新 HMI 才会同步显示

新增按钮：新增配方文件，用户选择本地配方文件上传

选择按钮：studio 使用当前配方文件

删除按钮：删除当前配方文件

保存按钮：修改配方文件后保存

配方文件修改方式：单击表格数据->修改

小组件组件 ​

样式部分就不一一缀叙

此类组件主要为装饰类组件，用户可自行选择，其数字计数组件、数字翻牌组件、倒计时组件绑定参数类型为 number 类型

​

图片组件 ​

样式部分就不一一缀叙

图像组件支持用户自定义上传图片充当组件

图片属性：

fill：将图片或视频拉伸或压缩，以填充容器，不保持宽高比。

contain：在保持宽高比的情况下，缩放图片，使其能够完全包含在容器中，容器可能会留出空白区域。

cover：在保持宽高比的情况下，缩放图片，使其能够完全覆盖容器，容器可能会有部分内容被裁剪。

scale-down：在保持宽高比的情况下，缩小图片，使其不超过容器的大小，如果缩小后的大小小于原始大小，则按原始大小显示

none：不对图片或视频进行任何调整，它们会按原始大小显示在容器中。

图标组件 ​

样式部分就不一一缀叙

图标组件目前内置五种图标供用户选择，分别为暂停图标、单步运行/启动图标、循环运行图标、设置图标、运行状态

这五大图标均与 studio 关联，控制着 studio 程序的运行，用户可直接拖入画布中使用，无需过多设置

图像组件 ​

样式部分就不一一缀叙

此组件也为 3D 图组件，用户可以选择不同 sceneID 展示不同 3D 图，用户可直接拖入画布中使用，无需过多设置
