工程页面总览 ​

工程页面整体布局分为组件选择、图层、画布、组件(画布)设置四大部分

组件选择区域 ​

组件分为图表组件、信息组件、列表组件、小组件组件、图片组件、图表组件、图像组件七大部分，组件选择区域内组件可以添加到画布当中，进行展示

1.展示组件 ​

展示组件选择区域组件支持三种方式

a. 组件直接拖拽放入画布区域

b. 双击组件

c.点击搜索组件，单击符合条件组件

双击以及单击方式添加组件，组件默认添加到画布区域右上角

2.组件列数 ​

列数支持单列以及双列，通过搜索框右侧按钮切换

a.双列

b.单列

3.显示分类 ​

组件类型只有一种时，默认隐藏分类，可以在系统设置页面显示分类(系统设置页面通过设置按钮打开，设置按钮在工程页面右上角)

图层区域 ​

图层区域主要作用为控制画布内组件，分为分组组件以及单组件，单击组件,右侧组件设置部分会出现相应组件设置

1.展示方式 ​

图层区域支持文本列表以及缩略图展示，通过图层区域顶部按钮切换

2.分组 ​

a.创建分组 ​

图层内创建分组方式为选中多个组件，右键->创建分组 或 CTRL+G 快捷键创建分组

b.解除分组 ​

分组文件夹右键->解除分组 或 选中分组后 CTRL+SHIFT+G 快捷键解除分组

画布中组件也可选中以及右键，多选快捷键为 CTRL+鼠标左键

3.右键功能 ​

组件/分组文件夹右键会显示相应右键功能

复制、剪切、粘贴相应快捷键分别为 CTRL+C 、CTRL+X 、CTRL+V

置顶、置底、上移、下移会改变组件在图层中位置

隐藏控制的是组件在画布中的显示与隐藏

锁定控制的是组件状态，组件锁定后画布内组件会处于不可选中状态，只能点击图层中组件来修改相应设置

隐藏与锁定除右键触发外，也可单击鼠标悬浮在图层组件中显现的按钮触发

分组内的组件，右键是整个分组的选项，而不是分组内单组件选项

画布区域 ​

画布区域分为画布以及底部按钮

1.画布 ​

画布为所有组件显示区域，单击组件或者画布右侧组件(画布)设置会显示相应设置，画布中只支持操作组件大小以及位置

操作组件位置: 鼠标左键按住组件移动 或 选中组件后 CTRL+ ⬆/⬇/⬅/➡

快捷键操作组件位置 移动距离在系统设置页面中设置

操作组件大小：选中组件后拉伸组件即可

画布缩放：CTRL+鼠标滚轮

画布位置：拖动画布区域右侧以及底部滚动条

2.底部按钮 ​

底部按钮从左到右依次为历史记录、历史记录提示、快捷键提示、画布缩放选项、锁定/不锁定画布缩放比、滑动选择画布缩放比列

历史记录会记录用户操作，最多记录 100 条

画布缩放选项可以控制画布缩放比列，提供了五个选项供用户选择，分别为自适应、50%、100%、150%、200%，其中选择自适应后画布会还原默认缩放比并自动居中

锁定画布比后会保持当前缩放比，CTRL+鼠标滚轮将不会改变画布缩放比，画布缩放选项以及滑动选择画布缩放比列功能都会禁用

组件(画布)设置区域 ​

1.画布设置 ​

右侧设置区域默认显示画布设置，点击画布也会显示画布设置，画布设置分为页面配置和主题颜色两大部分

a.页面配置 ​

画布背景图点击选择图片以及拖入图片(部分组件也可拖拽)选择

适配方式为画布在预览页展示方式，可以自行选择

滤镜为画布中全体组件增加滤镜效果，可以自行选择滤镜效果

b.主题颜色 ​

主题颜色为图表区域数据展示颜色，除了系统自带主题颜色，用户可点击自定义颜色，进入颜色配置页面，增加主题配色

2.组件配置 ​

组件配置分为定制、动画、数据、事件四大模块

a.定制模块 ​

定制模块主要是组件的一些显示设置

中间标出部分六个按钮作用为控制组件在画布中位置，从左至右依次为 靠左显示、y 方向居中显示、靠右显示、靠上显示、x 轴居中显示、靠下显示

滤镜/变化 可以配置当前组件的滤镜效果，其优先级大于画布设置中设置的滤镜效果

此处只介绍了公共设置，组件部分单独介绍请查阅常用组件

b.动画模块 ​

动画模块控制的是当前组件在预览页打开时的出场动画，内置了近五十种动画以供选择

分组中组件的动画依然有效

c.数据模块 ​

组件绑定 studio 参数后此模块配置无效

数据模块控制的是当前组件数据来源，分为动态请求与静态请求

静态请求 ​

静态请求可以直接导入 json/txt 文件，也可下载当前组件数据，下载文件格式为 json

动态请求 ​

动态请求主要分为请求配置、数据过滤部分

请求配置 ​

请求配置主要是对请求的一些配置，包含全局公共配置、组件单独配置

组件请求路径规则：全局公共配置请求路径+组件单独请求路径

更新间隔优先级：组件单独更新间隔>全局公共配置更新间隔

请求支持 GET、POST、PUT、PATCH、DELETE、SQL 等多种请求方式，用户可自行选择以及配置参数

数据过滤 ​

数据过滤部分主要是对请求返回的数据进行过滤，以返回当前组件需要的数据格式

此操作需要符合 Javascript 规范

d.事件模块 ​

事件模块支持用户为当前组件添加自定义事件，分为基础事件配置以及高级事件配置

事件配置需要符合 Javascript 规范
