版本说明 ​

0.3.2(2023-12-15) ​

New Features ​

  • Studio: 增加系统错误日志的打印
  • Studio: 增加读取点云和保存点云对ply、txt、csv格式点云的支持
  • Studio: 优化算子参数名称的翻译
  • Studio: 增加示例界面，并增加入门示例、进阶示例、工程示例
  • Studio: 增加与HMI工程的绑定，可以从Studio中直接打开当前的HMI工程
  • Studio: 优化用户操作后的系统弹窗内容
  • Studio: 优化UI界面和用户操作
  • HMI: 增加"图标组"组件，集成运行、停止、设置、状态等按钮
  • HMI: 增加Studio已支持图标的提示(组件左上角3个点)
  • HMI: 优化3D图像组件的显示，默认居中显示
  • HMI: 优化UI显示
  • HELPER: 增加AI-Vision软件的使用说明和示例介绍

Bug Fixes ​

  • Studio: 修复算子中寄存器和图像选择参数没有上限值的bug
  • Studio: 修复配方管理界面中添加列内容失败的bug
  • Studio: 修复试用时间结束点保存却是另存为的逻辑bug
  • Studio: 修复设备管理界面修改设备名称保存后提示错误的bug
  • HMI: 修复画布不跟随窗口自适应的bug
  • HMI: 修复打开旧工程文件找不到组件的bug
  • HMI: 修复组件分组后无法修改分组内组件参数的bug
  • HMI: 修复已知的显示bug

0.3.1(2023-12-7) ​

New Features ​

  • system: 增加配方管理界面
  • system: 增加系统日志的打印，路径为.log/run.log
  • system: 修改全局变量保存条件判断，用户选择“保留”时，保存工程会保存该全局变量
  • operator: 增加逻辑算子：判断(judge)
  • operator: 优化高度算子运行速度
  • operator: 增加"3D距离"算子的支持
  • operator: 修改"3D平面度"算子的参数，添加所有点计算模式
  • example: 增加"SRR5"示例工程
  • hmi: 修改导出工程的方式，可选择保存路径和名称，同时生成工程图片
  • hmi: 添加折线图的数据合并和钻取功能
  • hmi: 增加组件编号
  • hmi: 修改工程文件名后缀为aivh
  • hmi: 优化界面显示和图标

Bug Fixes ​

  • operator: 修复斑点算子输出结果错误的bug
  • operator: 修改"3D孔"的默认检测参数，修复找不到孔的bug
  • operator: 修复"3D裁切预处理"选择从寄存器获取区域类型参数，没有输出的bug
  • tool: 修复工具复制或剪切操作后，粘贴内容为空的bug
  • system: 修复加载错误配方格式时，软件异常退出的bug
  • system: 修复修复相机连接失败时，软件异常退出的bug
  • hmi: 修复图标无法显示的bug
  • ui: 修复已知的显示问题

0.2.5(2023-10-30) ​

New Features ​

  • system: 增加HMI功能
  • system: 增加点击帮助按钮弹出帮助界面的功能
  • system: 增加试用倒计时功能（90分钟）

  • device: 增加gocator同步取图相机数量支持最大为8个

  • operator: 增加加载点云算子对读取文件夹中所有文件的支持
  • operator: 增加IO算子：读取IO、写IO、等待IO信号
  • example: 增加"JST 3W"示例工程
  • example: 修改"196pin"示例工程，添加全局变量保存，并创建HMI工程
  • project: 增加工程文件的版本设置，工程文件版本要与软件版本一致
  • ui: 增加设备管理，包含Adlink板卡、gocator相机、lvm相机
  • ui: 增加全局变量管理，包含：创建、查询、修改

Bug Fixes ​

  • project: 修复工程文件后缀名错误或内容错误软件没有提示的bug
  • ui: 修复算子编辑界面不能拖到软件边缘的bug

  • ui: 修复已知显示bug
